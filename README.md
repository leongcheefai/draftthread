# Drafthread

AI-powered thread builder for **X (Twitter)** and **Threads**. Turn your ideas into viral threads with platform-specific formatting, character limits, and optimized post structures.

## Features

- **Multi-platform support** — Generate threads for X (280 char limit) or Threads (500 char limit)
- **Thread types** — Lesson, Launch, Milestone, Story, Listicle
- **Tone control** — Professional, Casual, Bold, Educational
- **Live preview** — Real-time platform-styled preview with character count warnings
- **Inline editing** — Edit individual posts directly in the preview
- **Copy & export** — Copy individual posts or full threads (plain text + Typefully format)
- **Responsive** — Desktop two-panel layout, mobile tab-based interface

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 18+
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

### Build

```bash
pnpm build
pnpm start
```
